# ☕ Party Watcher (Client)

Party Watcher is a real-time web application that allows users to create virtual lounges, share YouTube videos, and hang out with friends in an interactive, chat-driven environment.

The application focuses on responsive layouts, immediate user presence feedback, and a unified interactive workspace.

---

## ⚡ Technical Architecture & Implementation

Under the hood, the client is engineered around a modular, reliable frontend stack:

- **Real-Time Presence & Chat:** Orchestrated via **Socket.IO Client** to handle event-driven push architectures. The client listens for user entry/exit events and text streams, updating the active UI instantly without requiring page reloads or heavy polling loops.
- **Layout Isolation:** Engineered with clean Separation of Concerns (SoC). Dashboard hubs leverage a fluid, responsive card grid layout, while active rooms dynamically reorganize screen real estate to host an edge-to-edge video canvas alongside a live sidebar.
- **Declarative Route Guards:** Custom route layers intercept navigation requests, ensuring protected app areas handle asynchronous loading cycles gracefully and block unauthenticated access natively.
- **Type Safety:** Built entirely with strict **TypeScript** compilation path enforcement, ensuring data format contracts are fully respected across both HTTP (Axios) boundaries and live WebSocket channels.

---

## 🛠️ Tech Stack

- **Core Engine:** React 18 & TypeScript
- **Build Tooling:** Vite (Fast Hot Module Replacement)
- **State & Networking:** Axios (RESTful API Handshakes) & Socket.IO-Client (WebSockets)
- **Routing:** React Router Dom
- **Design System:** Tailwind CSS & Lucide Icons
